## OS ##
. 가상화 기술 : 가상의 컴퓨터 (= 가상 머신)
. 운영 체제를 설치 : Windows 10, Windows Server 2022, Linux
. Linux 역사
 - 1991년 리누스 토르발즈(Linus Torvalds) --> 0.01 --> 0.02 발표
 - 리차드 스톨만 : FSF(자유 소프트웨어 연맹) 1985년... C 컴파일러~
 - Open Source : 자유 --> 다운로드의 자유, 배포의 자유, 수정의 자유, 판매의 자유
 - 커널 (Kernel) : 자동차의 엔진
 - 배포판 : Redhat, Ubuntu, CentOS, Debian, Rocky......
 - RHEL(상용) >> CentOS >> Rocky 9.x

리눅스 기본 명령어
- 사용자 : 관리자(root), 일반사용자(rocky)
  사용자의 홈 디렉터리 (/home/사용자명), 관리자(/root)
- 폴더 구조
  C:\
   /
- cd : Change Directory. 디렉터리(=폴더) 이동
  pwd : Print Working Directory. 현재 디렉터리 확인
  ls : List. -l
  .. : 앞 디렉터리
  . : 현재 디렉터리
  whoami : 현 사용자 확인
  절대경로, 상대경로 :  /home/rocky
  mkdir : Make Directory
  rmdir : Remove Directory
  rm -rf : 한방에 지우기
  touch : 빈 파일 생성 or 생성시간을 현재로 변경
  cp 소스 타겟 : Copy. -r 폴더 복사
  mv 소스 타겟 : Move. 파일명 변경 or 파일 이동
  rm -f : 파일 삭제
  vi : Visual
  cat : 파일 내용 확인
  su - : 관리자로 접속
  # : 관리자, $ : 일반사용자
. 관리자 작업
 - 새 사용자 생성 : useradd
 - 사용자 제거 : userdel -r
 - 사용자 비번 지정 : passwd 사용자명
. su -사용자명 : 그 사용자로 접속
. 설치 : dnf -y install 프로그램명
